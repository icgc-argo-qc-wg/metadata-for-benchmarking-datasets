# Reference data directory structure
Directory structure of ARGO reference data files hosted at [Cancer Genome Collaboratory](https://cancercollaboratory.org/) in the `genomics-public-data` data store bucket.

```
genomics-public-data
├── gatk-resources
│   ├── 1000g_pon.hg38.vcf.gz
│   ├── 1000g_pon.hg38.vcf.gz.tbi

│   ├── af-only-gnomad.hg38.vcf.gz.tbi

│   ├── af-only-gnomad.pass-only.biallelic.snp.hg38.vcf.gz.tbi

│   ├── af-only-gnomad.pass-only.hg38.vcf.gz.tbi
│   ├── small_exac_common_3.hg38.vcf.gz
│   └── small_exac_common_3.hg38.vcf.gz.tbi
├── open-access-regions
│   └── open_access.gencode_v38.20210520.bed.gz
├── reference-genome
│   └── GRCh38_hla_decoy_ebv
│       ├── GRCh38_hla_decoy_ebv.dict
│       ├── GRCh38_hla_decoy_ebv.fa
```
